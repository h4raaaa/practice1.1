| **Анализ требований** | Сбор и фиксация потребностей заказчика. Интервью с заказчиком, анализ конкурентов, составление пользовательских сценариев | 01.03.2024 | 14.03.2024 |
| **Проектирование** | Архитектура системы, выбор технологического стека, проектирование интерфейсов, создание дизайн-макетов | 15.03.2024 | 11.04.2024 |

| **Тестирование** | Проверка корректности работы, функциональное и нагрузочное тестирование, поиск и исправление дефектов | 31.05.2024 | 20.06.2024 |
